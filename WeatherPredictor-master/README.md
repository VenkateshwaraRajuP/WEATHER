## Weather Predictor Frontend

This is a React + TypeScript frontend skeleton for a Weather application.

- **Goal**: Provide a scalable, production-ready folder structure.
- **Note**: Only boilerplate structure and placeholder components are implemented. No business logic, no API calls, and no API keys are included.

### Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Create a real `.env` file in the project root based on the guidance in `config/env.sample.md`.

### High-Level Structure

- `src/` – Application source
  - `components/` – Reusable and feature components
  - `pages/` – Top-level routed pages
  - `layouts/` – Layout components (shell, navigation, containers)
  - `routes/` – Routing configuration
  - `services/` – API service layer
  - `hooks/` – Reusable React hooks
  - `context/` – React context providers
  - `utils/` – Utility functions
  - `constants/` – App-wide constants and config
  - `assets/` – Static assets (images, icons, etc.)
  - `styles/` – Global styles
