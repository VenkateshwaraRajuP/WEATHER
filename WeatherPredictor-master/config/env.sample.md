## Environment Configuration (Sample)

This file documents the environment variables expected by the Weather Predictor frontend.
Create a `.env` file in the project root and copy these keys (with real values) when configuring the app.

```text
# Base URL for the weather API (e.g. https://api.openweathermap.org)
VITE_WEATHER_API_BASE_URL=

# API key for the weather API.
# Do NOT commit the real key. Keep it in your local .env only.
VITE_WEATHER_API_KEY=
```
