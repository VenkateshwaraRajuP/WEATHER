# Smart Weather Forecasting Using Machine Learning

Developed an AI-powered Weather Prediction System that analyzes historical weather data and predicts future weather conditions such as temperature, humidity, rainfall, and atmospheric pressure.

## 🚀 Key Features

- Predicts temperature, humidity, and rainfall
- Uses Machine Learning models for forecasting
- Analyzes historical weather datasets
- Provides real-time weather insights
- User-friendly dashboard for data visualization
- Supports data-driven decision making

## 🛠 Technologies Used

- Python
- Machine Learning
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
